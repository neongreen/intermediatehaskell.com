Intermediate Haskell

the book

This is a book about intermediate Haskell — parsing combinators, transformers and mtl, kinds and type algebra, GHC extensions, some advanced type-level concepts, libraries like lens, pipes and reflection, details of how data is represented in memory and how Haskell programs are evaluated, and so on. Basically, what professional Haskellers usually know, but people who have just finished LYAH usually don't.

Release date: beginning of 2018.

Written by Vladislav Zavialov and Artyom Kazak.
